\# ML Math Lab 🧠📈



> Machine Learning from first principles.  

> No black boxes. No blind `.fit()`. Just math, logic, and implementation.



\## Overview



\*\*ML Math Lab\*\* is a hands-on repository dedicated to understanding machine learning algorithms by building them from scratch using mathematics and core Python.



The goal is not just to \*use\* machine learning models, but to deeply understand:



\- how algorithms actually work

\- the mathematics behind optimization

\- why cost functions behave the way they do

\- how gradients update parameters

\- what assumptions each model makes

\- how decision boundaries emerge

\- where models fail



This repository avoids relying on high-level machine learning libraries for core implementations.



That means:



❌ No `sklearn.LinearRegression().fit()` magic  

❌ No treating ML like a vending machine  

✅ Pure mathematical implementation  

✅ Vectorized Python implementations  

✅ Derivations + intuition + experimentation
